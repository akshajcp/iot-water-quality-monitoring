# IoT Water Quality Monitoring System

## Overview
This project monitors water quality using sensors and Arduino. It measures parameters like pH, temperature, and turbidity.

## Components Used
- Arduino
- pH Sensor
- Temperature Sensor
- Turbidity Sensor
- Buzzer

## Features
- Real-time monitoring
- Alert system when values exceed limits

## Working
Sensors collect data → Arduino processes → alert is triggered if needed
